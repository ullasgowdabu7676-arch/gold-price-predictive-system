## 🚨 The Problem with Predicting Gold Prices

### 🔄 Volatile Markets
Gold prices fluctuate constantly due to:
- Global economic shifts  
- Currency changes  
- Geopolitical events  

➡️ Manual tracking becomes unreliable.

### ⚠️ Investor Risk
- Investors often rely on guesswork  
- Leads to unnecessary financial risk and missed opportunities  

### ❌ Manual Methods Fall Short
Traditional forecasting:
- Lacks precision  
- Cannot handle large datasets effectively

## 🎯 Project Objectives

### 1. Analyze Historical Data
Understand patterns in gold price time-series data.

### 2. Build a Predictive Model
Use machine learning to forecast future gold prices.

### 3. Support Investment Decisions
Provide insights to reduce risk and improve trade timing.

## 📊 Dataset Overview

The dataset contains historical gold price data across multiple years.

### Key Attributes:
- **Date** → Time index  
- **Gold Price** → Daily closing price  
- **Market Indicators** → Economic signals  

### 🎯 Target Variable:
- Future Gold Price

## 🤖 Algorithm Used: Linear Regression

### Why Linear Regression?

- ✔ Simple and interpretable  
- ✔ Works well with continuous data  
- ✔ Good baseline model  

### Benefits:
- Understand feature impact  
- Predict exact price values  

---

## 📈 Model Evaluation & Results

### Metrics Used:

- **MAE (Mean Absolute Error)**  
  Average prediction error  

- **RMSE (Root Mean Square Error)**  
  Penalizes large errors  

- **R² Score**  
  Measures model accuracy  

### Evaluation Method:
- Compare predicted vs actual prices  
- Use graphs and residual plots
